# CEWS
> CEWS is a **c**ode **e**ditor for mobile devices.

> Yes, the S stands for **s**crolling. How could you tell? :smirk:

## Built With
* [React Native](https://reactnative.dev)
* [CodeMirror](https://codemirror.net)


## Features

- Bracket matching
- Indentation
- Light/dark mode
- Search terms
- Syntax highlighting
- Themes
- Undo changes
